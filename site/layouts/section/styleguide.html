Interface Icons

We love the use of material design icons. Google material design provided a vast set of ready to use icon with simple copy and past code.

Application on Website

Step 1 - Link the stylesheet from google

<link href="https://fonts.googleapis.com/icon?family=Material+Icons" rel="stylesheet" />

Step 2 - Use the following css code to fix alignment issue. 

.material-icons {
  font-size: 1.2em;
  display: inline-flex;
  vertical-align: middle;
}

Step 3 - copy the code from material design icon browser and paste it at your desired location on the html. 

<div class="color-swatch" style="background-color: #81222e;">
  <span class="material-icons">content_copy</span>
  <span>#81222E</span>
</div>

Notes: if you are working with the hugo code base created for CaseIT 2021 webste, these code should be set up correctly already. No need to manually insert them on every page.
